# 开发计划

## 第 1 阶段：基础工程（已完成）

- 创建 React、TypeScript、Vite 和 Tailwind CSS 前端。
- 创建页面、组件和路由骨架。
- 创建 FastAPI 分层目录。
- 实现 `GET /health`。
- 建立项目文档和基础测试。

## 第 2 阶段：Mock 解题链路（已完成）

- 定义题目与解析的 Pydantic schema。
- 实现 mock `ai_solver`。
- 实现 `POST /api/problems/solve`。
- 在前端接入题目提交和完整解析展示。

## 第 3 阶段：SQLite 持久化（已完成）

- 建立题目、解析、标签和学习状态模型。
- 保存解题结果。
- 实现历史记录、详情、掌握状态和备注接口。

## 第 4 阶段：归纳与复盘（已完成）

- 按标签聚合题型。
- 增加掌握情况筛选和复盘视图。
- 完善教学分析展示。

## 第 5 阶段：DeepSeek Provider（已完成）

- 在保留 Mock 实现的前提下接入 DeepSeek。
- 增加配置校验、安全日志、错误处理和测试覆盖。

## 第 6 阶段：部署与持续集成

- 增加部署配置和持续集成。
- 增加生产环境数据库迁移和运行监控。
