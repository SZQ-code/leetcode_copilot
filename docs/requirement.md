# LeetCode Copilot 需求说明

## 产品目标

LeetCode Copilot 面向编程学习者，帮助用户完成从理解算法题到复盘知识点的完整学习闭环。

## MVP 功能范围

1. 输入算法题文本。
2. 调用 `POST /api/problems/solve` 获取结构化解析。
3. 使用 mock `ai_solver` 生成解析，不接入真实大模型。
4. 展示标题、难度、标签、总结、思路、算法选择原因、Python 代码、代码解释、复杂度、常见错误、边界情况和教学分析。
5. 使用 SQLite 保存题目和解析。
6. 查看历史记录和题目详情。
7. 按算法标签查看题型归纳。
8. 修改掌握状态和个人备注。

## 当前已实现范围

当前已创建前后端工程、页面和组件骨架、后端分层目录、健康检查接口及项目文档，并已实现：

- `POST /api/problems/solve`
- 固定“两数之和”mock `ai_solver`
- 题目文本校验
- 前端题目提交、加载、错误和完整解析展示
- SQLite/SQLAlchemy 题目、解析和标签模型
- 解题记录自动保存
- 历史记录和详情查询
- 掌握状态与个人备注更新

以下内容尚未实现：

- 数据库迁移
- 分类统计和复盘业务
- 真实大模型 API

## 非功能要求

- 前后端可独立安装和启动。
- 目录按职责拆分，方便后续测试和替换实现。
- 前端支持桌面端和移动端基础布局。
- 所有业务接口使用 JSON。
