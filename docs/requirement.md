# LeetCode Copilot 需求说明

## 产品目标

LeetCode Copilot 面向编程学习者，帮助用户完成从理解算法题到复盘知识点的完整学习闭环。

## MVP 功能范围

1. 输入算法题文本。
2. 调用 `POST /api/problems/solve` 获取结构化解析。
3. 使用 mock `ai_solver` 生成解析，不接入真实大模型。
4. 展示标题、难度、标签、总结、思路、算法选择原因、Python 代码、代码解释、复杂度、常见错误、边界情况和教学分析。
5. 使用 SQLite 保存题目和解析。
6. 查看历史记录和题目详情。
7. 按算法标签查看题型归纳。
8. 修改掌握状态和个人备注。

## 当前阶段范围

本阶段只创建前后端工程、页面和组件骨架、后端分层目录、健康检查接口及项目文档。

以下内容不在当前阶段实现：

- `POST /api/problems/solve`
- mock `ai_solver`
- SQLite 模型和迁移
- 历史记录、详情、分类和备注的实际业务
- 真实大模型 API

## 非功能要求

- 前后端可独立安装和启动。
- 目录按职责拆分，方便后续测试和替换实现。
- 前端支持桌面端和移动端基础布局。
- 所有业务接口使用 JSON。
